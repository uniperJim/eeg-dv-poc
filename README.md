# VeGAS EEG-DV — PoC (live demo)

This repository hosts **only the built static files** for the public demo:

> https://uniperjim.github.io/eeg-dv-poc/

It is an in-browser proof-of-concept (React, no backend, no server) for a
renewable-power Direktvermarktung workflow. Everything shown is **synthetic** —
invented parks, IDs, counterparties and coordinates; no real customer or master
data. Any file you import in the demo is parsed in your browser and never uploaded.

The application source is maintained privately and is not published here.
